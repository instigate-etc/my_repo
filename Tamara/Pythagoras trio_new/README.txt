2. INTRODUCTION
Pythagoras' triplet  is introduced below: 
1. a < b < c
2. a^2 + b^2 = c^2
E.g
3^2 + 4^2 = 9 + 16 = 25 = 5^2
3+4+5=12

The program is written in Java programming language.
This program worked for any numbers which you can import.
For the runing program you need to input some number.
